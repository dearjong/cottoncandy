## 다음에 만들면 좋은 문서 (순서 제안)

1. **IA (정보 구조)** — Admin 메뉴 트리, URL, 화면 그룹, 용어 통일.
2. **PRD (제품 요구)** — MVP 범위, 비-MVP(후순위), 성공 지표, 제약.
3. **릴리스/배포** — CI, 서버 경로, 환경 변수(PORT 등), 롤백 절차(팀 표준에 맞게).

`IA`·`PRD`는 팀 템플릿이 있으면 그 형식에 맞춰 이 허브에서 링크만 걸어도 됩니다.
