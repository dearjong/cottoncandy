# ADMarket 프로젝트 상태 관리 명세

---

### **1️⃣ 프로젝트 메인 상태 (Main Status)**

| 대분류 | 하위 상태 | 코드 | 설명 |
| --- | --- | --- | --- |
| **등록 (REGISTER)** | 임시저장 | `DRAFT` | 프로젝트 작성 중, 임시 저장 상태 |
|  | 승인요청 | `REQUESTED` | 광고주가 운영자에게 승인 요청한 상태 |
|  | 승인완료 | `APPROVED` | 운영자 승인 완료 |
|  | 접수중 | `PROPOSAL_OPEN` | 제안서 접수 및 참여 신청 가능한 상태 |
|  | 접수마감 | `PROPOSAL_CLOSED` | 제안 접수 기간이 종료된 상태 |
| **접수단계 (SUBMISSION)** | OT 예정 | `OT_SCHEDULED` | 오리엔테이션 일정 예정 |
|  | OT 완료 | `OT_COMPLETED` | 오리엔테이션 종료 |
|  | 제안서 제출 대기 | `PROPOSAL_SUBMIT` | 파트너가 제안서 준비 중 |
|  | 제안서 제출 완료 | `PROPOSAL_SUBMITTED` | 제안서 제출 완료 |
|  | PT 예정 | `PT_SCHEDULED` | 프리젠테이션 일정 확정 |
|  | PT 완료 | `PT_COMPLETED` | 프리젠테이션 완료 |
|  | 미선정 | `NO_SELECTION` | 참여사 미선정 (탈락) |
|  | 선정완료 | `SELECTED` | 최종 참여사 선정 완료 |
| **계약 (CONTRACT)** | 계약완료 | `CONTRACT` | 계약 체결 완료 |
| **제작 (PRODUCTION)** | 촬영 중 | `SHOOTING` | 영상 촬영 진행 중 |
|  | 후반 작업 | `EDITING` | 편집·CG·음향 등 후반 프로세스 |
|  | 시안 제출 | `DRAFT_SUBMITTED` | 시안 영상 제출됨 |
|  | 최종본 컨펌 완료 | `FINAL_APPROVED` | 최종 결과물 승인 완료 |
| **제작완료 (PRODUCTION_COMPLETE)** | — | `PRODUCTION_COMPLETE` | 제작 전체 완료 |
| **온에어 시작 (ONAIR_STARTED)** | — | `ONAIR_STARTED` | 광고 송출 시작 |
| **사후관리 (AFTER_SERVICE)** | — | `AFTER_SERVICE` | KPI 등록, 리뷰 관리 중 |
| **완료됨 (COMPLETE)** | — | `COMPLETE` | 프로젝트 종료 및 정산 완료 |
| **중단됨 (STOPPED)** | 관리자 확인중 | `ADMIN_CHECKING` | 중단 요청 검토 중 |
|  | 관리자 승인됨 | `ADMIN_CONFIRMED` | 중단 승인 완료 |
| **취소됨 (CANCELLED)** | 관리자 확인중 | `ADMIN_CHECKING` | 취소 요청 검토 중 |
|  | 관리자 승인됨 | `ADMIN_CONFIRMED` | 취소 승인 완료 |

### **2️⃣ 노출 상태 (Visibility Status)**

| 상태명 | 코드 | 설명 |
| --- | --- | --- |
| 공개 | `PUBLIC` | 모든 회원이 프로젝트를 볼 수 있음 |
| 비공개 | `PRIVATE` | 승인된 파트너만 열람 가능 |
| 초대 전용 | `INVITE_ONLY` | 초대받은 기업만 참여 가능 |
| 숨김 | `HIDDEN` | 운영자 및 내부 전용 상태 |

### **3️⃣ 파트너 참여 상태 (Partner Engagement)**

| 분류 | 하위 상태 | 코드 | 설명 |
| --- | --- | --- | --- |
| **참여 신청** | 참여신청 | `APPLY` | 파트너가 직접 참여 요청 |
| **초대** | 초대됨 | `INVITED` | 특정 파트너에게 초대 발송됨 |
|  | 미초대됨 | `NOT_INVITED` | 초대 대상 아님 |
| **OT (오리엔테이션)** | 참석확정 | `OT_ACCEPTED` | OT 일정 참석 확정 |
|  | 참석불가 | `OT_INVITED` | OT 초대 받았으나 불참 |
|  | 미참석 | `OT_NOT_ATTENDED` | OT에 불참 |
|  | 참석 완료 | `OT_ATTENDED` | OT 완료 |
| **제안서** | 제출 대기 | `PROPOSAL_SUBMIT` | 제안서 작성 중 |
|  | 제출 완료 | `PROPOSAL_SUBMITTED` | 제안서 제출 완료 |
| **PT (프리젠테이션)** | 참석확정 | `PT_ACCEPTED` | PT 일정 참석 확정 |
|  | 참석불가 | `PT_INVITED` | PT 초대 받았으나 불참 |
|  | 미참석 | `PT_NOT_ATTENDED` | PT에 불참 |
|  | 발표 완료 | `PT_COMPLETED` | PT 발표 완료 |
| **계약** | 계약완료 | `CONTRACTED` | 계약 체결 완료 |
| **완료** | 완료됨 | `COMPLETE` | 프로젝트 종료 |
| **중단** | 중단됨 | `STOPPED` | 진행 중단 |
| **취소** | 취소됨 | `CANCELLED` | 프로젝트 취소 |

### **4️⃣ 제작 진행 상태 (Production Status)**

| 상태명 | 코드 | 설명 |
| --- | --- | --- |
| 기획 진행 중 | `PLANNING` | 아이디어·스토리보드 단계 |
| 프리프로덕션 | `PPM` | 촬영 준비, 미팅 단계 |
| 촬영 중 | `SHOOTING` | 실제 촬영 진행 중 |
| 후반 작업 | `EDITING` | 편집, 음향, CG 등 후반 작업 |
| 최종 납품 | `DELIVERED` | 결과물 납품 완료 |
